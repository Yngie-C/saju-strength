# Toss 스토어 - 사주강점 (Saju-Strength) 등록

## 1. 앱 이름
사주강점

---

## 2. 한 줄 소개 (50자 이내)
타고난 기질 × 현재 강점 = 당신만의 성장 지도

---

## 3. 상세 설명 (300-500자)

### 앱 설명
사주강점은 동양의 전통 사주 분석과 현대적인 강점 진단을 결합한 유일한 서비스입니다.

생년월일시 입력만으로 당신의 사주 오행(목화토금수) 에너지 패턴을 분석하고, 60개의 정확한 질문으로 현재 발현 중인 강점을 측정합니다. 선천적 기질과 후천적 강점을 교차 분석하면 10가지 페르소나 유형 중 당신의 유형이 나타나고, 맞춤형 성장 로드맵을 제시합니다.

### 특징
- **5분 완료**: 빠른 분석, 즉시 결과
- **100% 무료**: 기본 분석은 완전 무료
- **AI 비용 제로**: 템플릿 기반 분석으로 빠르고 정확함
- **프리미엄 심층 리포트**: 4주 성장 계획과 개인 브랜딩 가이드
- **결과 공유**: SNS로 당신의 페르소나 공유

### 대상
20-30대 자기계발에 관심 있는 분, 성향 진단을 좋아하는 분, 사주에 관심 있는 분

## 5. 업데이트 노트 (v1.0.0 - First Release)

### v1.0.0 - 초판 출시

사주강점 서비스가 정식 출시되었습니다.

#### 주요 기능
- 사주 오행 분석: 생년월일시 기반 5행 에너지 패턴 분석
- Big5 강점 설문: 60문항 기반 정확한 강점 진단
- 교차 분석 리포트: 선천 기질 × 후천 강점 비교 분석
- 10가지 페르소나 유형: 당신만의 성장 페르소나 발견
- 기본 리포트: 100% 무료 제공
- 프리미엄 심층 리포트: 4주 성장 로드맵 + 개인 브랜딩 가이드
- 결과 공유: 페르소나 공유 기능

#### 개선 사항
- 모바일 최적화된 UI/UX
- 부드러운 애니메이션과 즉각적인 결과 확인
- 개인정보 보호 강화 (개인정보 암호화 저장)
- 토스 미니앱 환경 완벽 지원

#### 기술적 안정성
- 빠른 로딩 속도 (< 1초 결과 분석)
- 안정적인 DB 관리 (Supabase)
- 개인정보 암호화 저장 및 자동 삭제

#### 고지
이 서비스는 동양 전통 기질 분석과 현대 강점 진단의 융합 서비스이며, 의학적·심리학적 진단을 대체하지 않습니다.

## 추가 정보

### 서비스 특징 요약

| 구분 | 설명 |
|------|------|
| **설문 시간** | 약 5분 |
| **결과 제공** | 즉시 |
| **기본 리포트** | 100% 무료 |
| **프리미엄 리포트** | 1,900원 (선택사항) |
| **개인정보** | 암호화 저장, 서비스 탈퇴 시 자동 삭제 |
| **분석 방식** | AI 없음, 템플릿 기반 (빠르고 정확함) |

### 서비스 흐름

1. **생년월일 입력** (Step 1/4) - 1분
   - 양음력 선택
   - 성별 선택
   - 정확한 시간 입력 (선택사항)

2. **사주 미리보기** (Step 2/4) - 30초
   - 사주 오행 분석 결과
   - 천간·지지 조화도
   - 사주 강도 시각화

3. **강점 설문** (Step 3/4) - 3분
   - 60개 질문 (7점 리커트 척도)
   - 모바일 최적화 UI
   - 진행률 실시간 표시

4. **통합 결과 리포트** (Step 4/4) - 즉시
   - 당신의 페르소나 유형
   - 5축 교차 분석 차트
   - 성장 기회 포인트
   - 프리미엄 심층 리포트 옵션

### 10가지 페르소나 유형

사주의 5행(목화토금수) × 4가지 강점 매트릭스(일치/잠재력/후천개발/미발현)로 도출되는 10가지 유형:

- **일치형**: 타고난 재능이 현재도 빛나는 영역
- **잠재력형**: 아직 발현되지 않은 숨은 보석
- **후천개발형**: 노력으로 만들어낸 강점
- **미발현형**: 앞으로의 성장 기회

### 프리미엄 심층 리포트 (1,900원)

- 강점 × 약점 상세 분석
- 4주 단계별 성장 로드맵
- 개인 브랜딩 가이드
- 커리어 및 대인관계 인사이트
- 다운로드 가능한 PDF 리포트
